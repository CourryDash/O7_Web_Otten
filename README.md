# O7 Coffee Shop - E-Commerce Platform

Project Full-Stack JavaScript untuk memenuhi tugas UAS Backend dan Frontend.

## 📁 Struktur Folder
* **Backend/**: Menggunakan express.js, node.js, dan lain sebagainya
* **Frontend/**: Menggunakan React.js.

## 🚀 Cara Menjalankan Proyek Lokal

### 1. Backend
1. Masuk ke folder backend: `cd Backend`
2. Install dependency: `npm install`
3. Duplikat file `.env.example` menjadi `.env` dan sesuaikan konfigurasi database Anda.
4. Jalankan server: `npm run dev` atau `node server.js`

### 2. Frontend
1. Masuk ke folder frontend: `cd Frontend`
2. Install dependency: `npm install`
3. Jalankan aplikasi: `npm start` atau `npm run dev`
